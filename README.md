
#### 爬取微信好友头像，并拼接成一张图片

#### 项目初始化

```

# clone 项目到本地

git clone https://github.com/liutaochange/python.git

# 安装项目依赖

pip install itchat
pip install pillow

# 到项目目录下，执行Python命令，运行程序

cd python
python search.py

```

#### 执行完 python search.py 会出现二维码，用微信扫一扫获取用户信息，就开始去获取微信好友头像，最后生成的图片，会保存在当前目录下一个saveImg的文件夹
